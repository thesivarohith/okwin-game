# 🎯 OkWin Game — Color Prediction AI

An AI-powered prediction engine that analyzes patterns in an online color prediction game, achieving a **60% accuracy rate**.

## 🧠 What It Does
- Observes and analyzes color game outcomes
- Uses pattern recognition and probability modeling to predict next results
- Achieves 60% accuracy — statistically above random chance (33%)

## ⚙️ Tech Stack
![Python](https://img.shields.io/badge/Python-3776AB?style=for-the-badge&logo=python&logoColor=white)

## 🚀 How It Works
1. Data is fed from the color game platform
2. The algorithm analyzes historical patterns
3. Prediction is generated for the next round

## ⚠️ Disclaimer
This project is built purely for **educational and research purposes** — to explore pattern recognition and probability in real-world data streams.
